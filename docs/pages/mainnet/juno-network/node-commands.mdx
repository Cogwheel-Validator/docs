# Usefull commands for Juno Network

## üîë Key management


<Callout type="info" emoji="üëõ">
  Change wallet in the commands to your own wallet name.
</Callout>

#### Add new key

```bash
junod keys add wallet
```

#### Recover existing key

```bash
junod keys add wallet --recover
```

#### List all keys

```bash
junod keys list
```

#### Delete key

```bash
junod keys delete wallet
```

#### Export key to the file

```bash
junod keys export wallet
```

#### Import key from the file

```bash
junod keys import wallet wallet.backup
```

#### Query wallet balance

```bash
junod q bank balances $( junod keys show wallet -a)
```

## üßô Validator management



<Callout type="info" emoji="üôã‚Äç‚ôÄÔ∏è">
  Make sure to change values for moniker, identity, details website to your own values.
</Callout>

#### Edit existing validator

```bash
junod tx staking edit-validator \
--new-moniker "YOUR_MONIKER_NAME" \
--identity "YOUR_KEYBASE_ID" \
--details "YOUR_DETAILS" \
--website "YOUR_WEBSITE_URL" \
--chain-id juno-1 \
--commission-rate 0.05 \
--from wallet \
--gas-adjustment 1.5 \
--gas auto \
--gas-prices 0.075ujuno \
-y
```

#### Unjail validator

```bash
junod tx slashing unjail --from wallet --chain-id juno-1 --gas-adjustment 1.5 --gas auto
 --gas-prices 0.075ujuno -y
```

#### Jail reason

```bash
junod query slashing signing-info $( junod tendermint show-validator)
```

#### List all active validators

```bash
junod q staking validators -oj --limit=3000 | jq '.validators[] | select(.status=="BOND_STATUS_BONDED")' | jq -r '(.tokens|tonumber/pow(10; 6)|floor|tostring) + " \t " + .description.moniker' | sort -gr | nl
```

#### List all inactive validators

```bash
junod q staking validators -oj --limit=3000 | jq '.validators[] | select(.status=="BOND_STATUS_UNBONDED")' | jq -r '(.tokens|tonumber/pow(10; 6)|floor|tostring) + " \t " + .description.moniker' | sort -gr | nl
```

#### View validator details

```bash
junod q staking validator $( junod keys show wallet --bech val -a)
```

## ü™ô Token management

#### Withdraw rewards from all validators

```bash
junod tx distribution withdraw-all-rewards --from wallet --chain-id juno-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.075ujuno -y
```

#### Withdraw commission and rewards from your validator

```bash
junod tx distribution withdraw-rewards $( junod keys show wallet --bech val -a) --commission --from wallet --chain-id juno-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.075ujuno -y
```

#### Delegate tokens to yourself

```bash
junod tx staking delegate $( junod keys show wallet --bech val -a) 1000000000000.075ujuno --from wallet --chain-id juno-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.075ujuno -y
```

#### Delegate tokens to validator

```bash
junod tx staking delegate <TO_VALOPER_ADDRESS> 1000000000000.075ujuno --from wallet --chain-id juno-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.075ujuno  -y
```

#### Redelegate tokens to another validator

```bash
junod tx staking redelegate $( junod keys show wallet --bech val -a) <TO_VALOPER_ADDRESS> 1000000$ppica --from wallet --chain-id juno-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.075ujuno -y
```

#### Unbond tokens from your validator

```bash
junod tx staking unbond $( junod keys show wallet --bech val -a) 1000000$ppica --from wallet --chain-id juno-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.075ujuno -y
```

#### Send tokens to the wallet

```bash
junod tx bank send wallet <TO_WALLET_ADDRESS> 1000000$ppica --from wallet --chain-id juno-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.075ujuno -y
```

## ‚ö°Ô∏è Utility

#### Update Indexer

**Disable indexer**

```bash
sed -i -e 's|^indexer *=.*|indexer = "null"|' $HOME/.banksy/config/config.toml
```

**Enable indexer**

```bash
sed -i -e 's|^indexer *=.*|indexer = "kv"|' $HOME/.banksy/config/config.toml
```

#### Update pruning

```bash
sed -i \
  -e 's|^pruning *=.*|pruning = "custom"|' \
  -e 's|^pruning-keep-recent *=.*|pruning-keep-recent = "2000"|' \
  -e 's|^pruning-keep-every *=.*|pruning-keep-every = "0"|' \
  -e 's|^pruning-interval *=.*|pruning-interval = "100"|' \
  $HOME/.banksy/config/app.toml
```

## üîß Maintenance

#### Get validator info

```bash
junod status 2>&1 | jq .ValidatorInfo
```

#### Get sync info

```bash
junod status 2>&1 | jq .SyncInfo
```

#### Enable prometheus

```bash
sed -i -e "s/prometheus = false/prometheus = true/" $HOME/.banksy/config/config.toml
```

#### Reset chain data

```bash
junod tendermint unsafe-reset-all --keep-addr-book --home $HOME/.banksy --keep-addr-book
```

#### Remove node

<Callout type="error" emoji="‚ò¢Ô∏è">
  Make sure you have backed up your priv_validator_key.json!
</Callout>

```bash
cd $HOME
sudo systemctl stop junod
sudo systemctl disable junod
sudo rm /etc/systemd/system/junod.service
sudo systemctl daemon-reload
rm -f $(which junod)
rm -rf $HOME/.banksy
rm -rf $HOME/composable-juno
```
