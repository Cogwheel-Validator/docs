# Usefull commands for Union Protocol

## üîë Key management


<Callout type="info" emoji="üëõ">
  Change wallet in the commands to your own wallet name.
</Callout>

#### Add new key

```bash
uniond keys add wallet
```

#### Recover existing key

```bash
uniond keys add wallet --recover
```

#### List all keys

```bash
uniond keys list
```

#### Delete key

```bash
uniond keys delete wallet
```

#### Export key to the file

```bash
uniond keys export wallet
```

#### Import key from the file

```bash
uniond keys import wallet wallet.backup
```

#### Query wallet balance

```bash
uniond q bank balances $( uniond keys show wallet -a)
```

## üßô Validator management



<Callout type="info" emoji="üôã‚Äç‚ôÄÔ∏è">
  Make sure to change values for moniker, identity, details website to your own values.
</Callout>

#### Edit existing validator

Edit the validator.json. Then execute the command.

```json
--moniker "YOUR_MONIKER_NAME" \
--identity "YOUR_KEYBASE_ID" \
--details "YOUR_DETAILS" \
--website "YOUR_WEBSITE_URL" \
--chain-id union-testnet-6 \
--commission-rate 0.05 \
--from wallet
```

```bash
uniond tx staking edit-validator /path/to/validator.json --gas-adjustment 1.5 --gas auto
 --gas-prices 0umuno -y
```

#### Unjail validator

```bash
uniond tx slashing unjail --from wallet --chain-id union-testnet-6 --gas-adjustment 1.5 --gas auto
 --gas-prices 0umuno -y
```

#### Jail reason

```bash
uniond query slashing signing-info $( uniond tendermint show-validator)
```

#### List all active validators

```bash
uniond q staking validators -oj --limit=3000 | jq '.validators[] | select(.status=="BOND_STATUS_BONDED")' | jq -r '(.tokens|tonumber/pow(10; 6)|floor|tostring) + " \t " + .description.moniker' | sort -gr | nl
```

#### List all inactive validators

```bash
uniond q staking validators -oj --limit=3000 | jq '.validators[] | select(.status=="BOND_STATUS_UNBONDED")' | jq -r '(.tokens|tonumber/pow(10; 6)|floor|tostring) + " \t " + .description.moniker' | sort -gr | nl
```

#### View validator details

```bash
uniond q staking validator $( uniond keys show wallet --bech val -a)
```

## ü™ô Token management

#### Withdraw rewards from all validators

```bash
uniond tx distribution withdraw-all-rewards --from wallet --chain-id union-testnet-6 --gas-adjustment 1.5 --gas auto --gas-prices 0umuno -y
```

#### Withdraw commission and rewards from your validator

```bash
uniond tx distribution withdraw-rewards $( uniond keys show wallet --bech val -a) --commission --from wallet --chain-id union-testnet-6 --gas-adjustment 1.5 --gas auto --gas-prices 0umuno -y
```

#### Delegate tokens to yourself

```bash
uniond tx staking delegate $( uniond keys show wallet --bech val -a) 1000000000000umuno --from wallet --chain-id union-testnet-6 --gas-adjustment 1.5 --gas auto --gas-prices 0umuno -y
```

#### Delegate tokens to validator

```bash
uniond tx staking delegate <TO_VALOPER_ADDRESS> 1000000000000umuno --from wallet --chain-id union-testnet-6 --gas-adjustment 1.5 --gas auto --gas-prices 0umuno  -y
```

#### Redelegate tokens to another validator

```bash
uniond tx staking redelegate $( uniond keys show wallet --bech val -a) <TO_VALOPER_ADDRESS> 1000000$uunion --from wallet --chain-id union-testnet-6 --gas-adjustment 1.5 --gas auto --gas-prices 0umuno -y
```

#### Unbond tokens from your validator

```bash
uniond tx staking unbond $( uniond keys show wallet --bech val -a) 1000000$uunion --from wallet --chain-id union-testnet-6 --gas-adjustment 1.5 --gas auto --gas-prices 0umuno -y
```

#### Send tokens to the wallet

```bash
uniond tx bank send wallet <TO_WALLET_ADDRESS> 1000000$uunion --from wallet --chain-id union-testnet-6 --gas-adjustment 1.5 --gas auto --gas-prices 0umuno -y
```

## ‚ö°Ô∏è Utility

#### Update Indexer

**Disable indexer**

```bash
sed -i -e 's|^indexer *=.*|indexer = "null"|' $HOME/.union/config/config.toml
```

**Enable indexer**

```bash
sed -i -e 's|^indexer *=.*|indexer = "kv"|' $HOME/.union/config/config.toml
```

#### Update pruning

```bash
sed -i \
  -e 's|^pruning *=.*|pruning = "custom"|' \
  -e 's|^pruning-keep-recent *=.*|pruning-keep-recent = "2000"|' \
  -e 's|^pruning-keep-every *=.*|pruning-keep-every = "0"|' \
  -e 's|^pruning-interval *=.*|pruning-interval = "100"|' \
  $HOME/.union/config/app.toml
```

## üîß Maintenance

#### Get validator info

```bash
uniond status 2>&1 | jq .ValidatorInfo
```

#### Get sync info

```bash
uniond status 2>&1 | jq .SyncInfo
```

#### Enable prometheus

```bash
sed -i -e "s/prometheus = false/prometheus = true/" $HOME/.union/config/config.toml
```

#### Reset chain data

```bash
uniond tendermint unsafe-reset-all --keep-addr-book --home $HOME/.union --keep-addr-book
```

#### Remove node



<Callout type="error" emoji="‚ò¢Ô∏è">
  Make sure you have backed up your priv_validator_key.json!
</Callout>

```bash
cd $HOME
sudo systemctl stop uniond
sudo systemctl disable uniond
sudo rm /etc/systemd/system/uniond.service
sudo systemctl daemon-reload
rm -f $(which uniond)
rm -rf $HOME/.union
rm -rf $HOME/unionchain
```
