# Usefull commands for Initia Testnet

## üîë Key management


<Callout type="info" emoji="üëõ">
  Change wallet in the commands to your own wallet name.
</Callout>

#### Add new key

```bash
initiad keys add wallet
```

#### Recover existing key

```bash
initiad keys add wallet --recover
```

#### List all keys

```bash
initiad keys list
```

#### Delete key

```bash
initiad keys delete wallet
```

#### Export key to the file

```bash
initiad keys export wallet
```

#### Import key from the file

```bash
initiad keys import wallet wallet.backup
```

#### Query wallet balance

```bash
initiad q bank balances $( initiad keys show wallet -a)
```

## üßô Validator management



<Callout type="info" emoji="üôã‚Äç‚ôÄÔ∏è">
  Make sure to change values for moniker, identity, details website to your own values.
</Callout>

#### Edit existing validator

```bash
initiad tx staking edit-validator \
--new-moniker "YOUR_MONIKER_NAME" \
--identity "YOUR_KEYBASE_ID" \
--details "YOUR_DETAILS" \
--website "YOUR_WEBSITE_URL" \
--chain-id initiation-1 \
--commission-rate 0.05 \
--from wallet \
--gas-adjustment 1.5 \
--gas auto \
--gas-prices 0.015uinit \
-y
```

#### Unjail validator

```bash
initiad tx slashing unjail --from wallet --chain-id initiation-1 --gas-adjustment 1.5 --gas auto
 --gas-prices 0.015uinit -y
```

#### Jail reason

```bash
initiad query slashing signing-info $( initiad tendermint show-validator)
```

#### List all active validators

```bash
initiad q staking validators -oj --limit=3000 | jq '.validators[] | select(.status=="BOND_STATUS_BONDED")' | jq -r '(.tokens|tonumber/pow(10; 6)|floor|tostring) + " \t " + .description.moniker' | sort -gr | nl
```

#### List all inactive validators

```bash
initiad q staking validators -oj --limit=3000 | jq '.validators[] | select(.status=="BOND_STATUS_UNBONDED")' | jq -r '(.tokens|tonumber/pow(10; 6)|floor|tostring) + " \t " + .description.moniker' | sort -gr | nl
```

#### View validator details

```bash
initiad q staking validator $( initiad keys show wallet --bech val -a)
```

## ü™ô Token management

#### Withdraw rewards from all validators

```bash
initiad tx distribution withdraw-all-rewards --from wallet --chain-id initiation-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.015uinit -y
```

#### Withdraw commission and rewards from your validator

```bash
initiad tx distribution withdraw-rewards $( initiad keys show wallet --bech val -a) --commission --from wallet --chain-id initiation-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.015uinit -y
```

#### Delegate tokens to yourself

```bash
initiad tx staking delegate $( initiad keys show wallet --bech val -a) 1000000uinit --from wallet --chain-id initiation-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.015uinit -y
```

#### Delegate tokens to validator

```bash
initiad tx staking delegate <TO_VALOPER_ADDRESS> 1000000uinit --from wallet --chain-id initiation-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.015uinit  -y
```

#### Redelegate tokens to another validator

```bash
initiad tx staking redelegate $( initiad keys show wallet --bech val -a) <TO_VALOPER_ADDRESS> 1000000uinit --from wallet --chain-id initiation-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.015uinit -y
```

#### Unbond tokens from your validator

```bash
initiad tx staking unbond $( initiad keys show wallet --bech val -a) 1000000uinit --from wallet --chain-id initiation-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.015uinit -y
```

#### Send tokens to the wallet

```bash
initiad tx bank send wallet <TO_WALLET_ADDRESS> 1000000uinit --from wallet --chain-id initiation-1 --gas-adjustment 1.5 --gas auto --gas-prices 0.015uinit -y
```

## ‚ö°Ô∏è Utility

#### Update Indexer

**Disable indexer**

```bash
sed -i -e 's|^indexer *=.*|indexer = "null"|' $HOME/.initia/config/config.toml
```

**Enable indexer**

```bash
sed -i -e 's|^indexer *=.*|indexer = "kv"|' $HOME/.initia/config/config.toml
```

#### Update pruning

```bash
sed -i \
  -e 's|^pruning *=.*|pruning = "custom"|' \
  -e 's|^pruning-keep-recent *=.*|pruning-keep-recent = "2000"|' \
  -e 's|^pruning-keep-every *=.*|pruning-keep-every = "0"|' \
  -e 's|^pruning-interval *=.*|pruning-interval = "100"|' \
  $HOME/.initia/config/app.toml
```

## üîß Maintenance

#### Get validator info

```bash
initiad status 2>&1 | jq .ValidatorInfo
```

#### Get sync info

```bash
initiad status 2>&1 | jq .SyncInfo
```

#### Enable prometheus

```bash
sed -i -e "s/prometheus = false/prometheus = true/" $HOME/.initia/config/config.toml
```

#### Reset chain data

```bash
initiad tendermint unsafe-reset-all --keep-addr-book --home $HOME/.initia --keep-addr-book
```

#### Remove node

<Callout type="error" emoji="‚ò¢Ô∏è">
  Make sure you have backed up your priv_validator_key.json!
</Callout>

```bash
cd $HOME
sudo systemctl stop initiad
sudo systemctl disable initiad
sudo rm /etc/systemd/system/initiad.service
sudo systemctl daemon-reload
rm -f $(which initiad)
rm -rf $HOME/.initia
rm -rf $HOME/initia
```
