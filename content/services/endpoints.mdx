# Endpoint 

Public blockchain endpoint are available for all mainnets. Not all testnets
have endpoint publicl so keep that in mind. Status of endpoint can be seen on the status page
https://status.cogwheel.zone

## Mainnet endpoints

### Dyson Protocol

| Dyson Protocol     | Links                            |
| :------------      | :----------:                     |
| REST API           | https://dyson-api.cogwheel.zone  |
| RPC                | https://dyson-rpc.cogwheel.zone  |
| GRPC               | grpc://dyson-grpc.cogwheel.zone:443  |
