# Node setup for Elys Network

Node setup requires some basic knowledge of Linux OS and server management. This guide will assume
you are using a Debian based OS such as Ubuntu for example.

## Update the system and install the essentials:

```bash
sudo apt update && sudo apt full-upgrade -y && apt install build-essential 
sudo apt install vim # or install nano
sudo apt install entr jq make git ufw wget 
```

## Install GO and Cosmovisor

```bash
sudo rm -r /usr/local/go/
wget https://go.dev/dl/gogo1.24.5.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf gogo1.24.5.linux-amd64.tar.gz
rm gogo1.24.5.linux-amd64.tar.gz

# Add GO to PATH
echo 'export GO111MODULE=on' >> $HOME/.bashrc
echo 'export GOPATH=$HOME/go' >> $HOME/.bashrc
echo 'export GOROOT=/usr/local/go' >> $HOME/.bashrc
echo 'export PATH=$PATH:$GOPATH/bin' >> $HOME/.bashrc
echo 'export PATH=$PATH:$GOROOT/bin' >> $HOME/.bashrc

# Source the bashrc file
source $HOME/.bashrc

# Install cosmovisor
go install cosmossdk.io/tools/cosmovisor/cmd/cosmovisor@vlatest
sudo mv ~/go/bin/cosmovisor /usr/local/bin
```

## Clone the repository and install node binary

```bash
cd $HOME 
git clone https://github.com/elys-network/elys
cd elys  # always doublecheck from the official team for the right version
git checkout v6.6.0
make install
```

## Set up cosmovisor

```bash 
mkdir -p $HOME/.elys/cosmovisor/genesis/bin
mkdir -p $HOME/.elys/cosmovisor/upgrades
mv $HOME/go/bin/elysd $HOME/.elys/cosmovisor/genesis/bin/
sudo ln -s $HOME/.elys/cosmovisor/genesis $HOME/.elys/cosmovisor/current -f
sudo ln -s $HOME/.elys/cosmovisor/current/bin/elysd /usr/local/bin/elysd -f

# Create service file

sudo tee /etc/systemd/system/elysd.service > /dev/null <<EOF
[Unit]
Description=Elys Network node"
After=network-online.target

[Service]
User=$USER
ExecStart=$(which cosmovisor) run start
WorkingDirectory=$HOME/.elys
Restart=on-failure 
RestartSec=5
LimitNOFILE=65535
Environment="DAEMON_NAME=elysd"
Environment="DAEMON_HOME=$HOME/.elys"
Environment="DAEMON_ALLOW_DOWNLOAD_BINARIES=false"
Environment="DAEMON_RESTART_AFTER_UPGRADE=true"
Environment="UNSAFE_SKIP_BACKUP=true"
Environment="PATH=$PATH:$HOME/.elys/cosmovisor/current/bin"

NoNewPrivileges=true
ProtectSystem=strict
RestrictSUIDSGID=true
LockPersonality=true
PrivateDevices=true
PrivateTmp=true
ProtectControlGroups=yes
ProtectKernelModules=yes
ProtectKernelTunables=yes
RestrictNamespaces=yes

[Install]
WantedBy=multi-user.target
EOF

sudo systemctl daemon-reload
sudo systemctl enable elysd.service
```

## Set up the node 

```bash
# Please when using this guide replace â€œYOUR_MONIKER" with name for your node.

elysd init YOUR_MONIKER --chain-id elys-1

# Get address book and genesis.json
curl -L https://files.cogwheel.zone/elys/genesis.json |  tar -xvf elys_genesis.tar.xz -C $HOME/.elys/config/genesis.json
wget https://files.cogwheel.zone/elys/addrbook.json -O $HOME/.elys/config/addrbook.json

#Set minimum gas prices
sed -i -e "s|^minimum-gas-prices *=.*|minimum-gas-prices = \"\"|" $HOME/.elys/config/app.toml
```
