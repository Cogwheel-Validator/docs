# Usefull commands for Juno Network

## üîë Key management

<Callout type="info" text={`Change wallet in the commands to your own wallet name.`} />

#### Add new key

```bash
junod keys add wallet
```

#### Recover existing key

```bash
junod keys add wallet --recover
```

#### List all keys

```bash
junod keys list
```

#### Delete key

```bash
junod keys delete wallet
```

#### Export key to the file

```bash
junod keys export wallet
```

#### Import key from the file

```bash
junod keys import wallet wallet.backup
```

#### Query wallet balance

```bash
junod q bank balances $( junod keys show wallet -a)
```

## üßô Validator management

<Callout type="info" text={`Make sure to change values for moniker, identity, details website to your own values.`} />

#### Edit existing validator

When editing a validator, you can use the following command. But you do not need to edit all the fields. 
You can edit only the fields you want to change. The gas flags are required for the command to work.
```bash
junod tx staking edit-validator \
--new-monike "YOUR_MONIKER_NAME" \
--identity "YOUR_KEYBASE_ID" \
--details "YOUR_DETAILS" \
--website "YOUR_WEBSITE_URL" \
--chain-id uni-7 \
--commission-rate "0.05" \
--gas-adjustment 1.5 \
--gas auto \
--gas-prices "" \
--from wallet
```

```bash
junod tx staking edit-validator /path/to/validator.json --gas-adjustment 1.5 --gas auto
 --gas-prices  -y
```

#### Unjail validator

```bash
junod tx slashing unjail --from wallet --chain-id uni-7 --gas-adjustment 1.5 --gas auto
 --gas-prices  -y
```

#### Jail reason

```bash
```bash
junod query slashing signing-info $( junod comet show-validator)
```

#### List all active validators

```bash
junod q staking validators -oj --limit=3000 | jq '.validators[] | select(.status=="BOND_STATUS_BONDED")' | jq -r '(.tokens|tonumber/pow(10; 6)|floor|tostring) + " \t " + .description.moniker' | sort -gr | nl
```

#### List all inactive validators

```bash
junod q staking validators -oj --limit=3000 | jq '.validators[] | select(.status=="BOND_STATUS_UNBONDED")' | jq -r '(.tokens|tonumber/pow(10; 6)|floor|tostring) + " \t " + .description.moniker' | sort -gr | nl
```

#### View validator details

```bash
junod q staking validator $( junod keys show wallet --bech val -a)
```

## ü™ô Token management

#### Withdraw rewards from all validators

```bash
junod tx distribution withdraw-all-rewards --from wallet --chain-id uni-7 --gas-adjustment 1.5 --gas auto --gas-prices  -y
```

#### Withdraw commission and rewards from your validator

```bash
junod tx distribution withdraw-rewards $( junod keys show wallet --bech val -a) --commission --from wallet --chain-id uni-7 --gas-adjustment 1.5 --gas auto --gas-prices  -y
```

#### Delegate tokens to yourself

```bash
junod tx staking delegate $( junod keys show wallet --bech val -a) 100000000000 --from wallet --chain-id uni-7 --gas-adjustment 1.5 --gas auto --gas-prices  -y
```

#### Delegate tokens to validator

```bash
junod tx staking delegate <TO_VALOPER_ADDRESS> 100000000000 --from wallet --chain-id uni-7 --gas-adjustment 1.5 --gas auto --gas-prices   -y
```

#### Redelegate tokens to another validator

```bash
junod tx staking redelegate $( junod keys show wallet --bech val -a) <TO_VALOPER_ADDRESS> 1000000$ublit --from wallet --chain-id uni-7 --gas-adjustment 1.5 --gas auto --gas-prices  -y
```

#### Unbond tokens from your validator

```bash
junod tx staking unbond $( junod keys show wallet --bech val -a) 1000000$ublit --from wallet --chain-id uni-7 --gas-adjustment 1.5 --gas auto --gas-prices  -y
```

#### Send tokens to the wallet

```bash
junod tx bank send wallet <TO_WALLET_ADDRESS> 1000000$ublit --from wallet --chain-id uni-7 --gas-adjustment 1.5 --gas auto --gas-prices  -y
```

## ‚ö°Ô∏è Utility

#### Update Indexer

**Disable indexer**

```bash
sed -i -e 's|^indexer *=.*|indexer = "null"|' $HOME/.juno/config/config.toml
```

**Enable indexer**

```bash
sed -i -e 's|^indexer *=.*|indexer = "kv"|' $HOME/.juno/config/config.toml
```

#### Update pruning

```bash
sed -i \
  -e 's|^pruning *=.*|pruning = "custom"|' \
  -e 's|^pruning-keep-recent *=.*|pruning-keep-recent = "2000"|' \
  -e 's|^pruning-interval *=.*|pruning-interval = "100"|' \
  $HOME/.juno/config/app.toml
```

## üîß Maintenance

#### Get validator info

```bash
junod status 2>&1 | jq .ValidatorInfo
```

#### Get sync info

```bash
junod status 2>&1 | jq .SyncInfo
```

#### Enable prometheus

```bash
sed -i -e "s/prometheus = false/prometheus = true/" $HOME/.blit/config/config.toml
```

#### Reset chain data

```bash
junod tendermint unsafe-reset-all --keep-addr-book --home $HOME/.blit --keep-addr-book
```

#### Remove node



<Callout type="error" emoji="‚ò¢Ô∏è">
  Make sure you have backed up your priv_validator_key.json!

```bash
